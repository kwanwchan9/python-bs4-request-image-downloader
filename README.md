# Introduction

This is a simple image downloader for download all image from URL webpage created by python with Requests and Beautiful Soup library for web scraping.

# Instruction

1. Clone the project and change the `URL`, `savePath` and `directory` for your own customization respectively.

   `url = "https://www.google.com"`
   `savePath = r"C:\Users\%USERPROFILE%\Desktop"`
   `directory = "New folder"`
